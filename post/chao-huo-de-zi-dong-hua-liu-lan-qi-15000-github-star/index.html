超火的自动化浏览器，15000+ GitHub Star！

2025-11-13 1 min read # AI

以前写浏览器自动化脚本的时候，经常被一个问题折磨：辛辛苦苦写好的脚本，刚过几天，就突然间无法运行。然后一查发现，爬取的网站又进行了改版，之前代码里写的定位元素全变，接着改代码、测试等，这种情况极其痛苦。

无独有偶，近日在 GitHub 上找到一个开源的自动化浏览器  Skyvern，一看竟然斩获了 15000+ GitHub Star。

用 AI 理解网页，不靠固定选择器以前传统的浏览器自动化工具，大部分都是使用 Selenium，需要我们精确指定每个元素的位置。

而 Skyvern 的做法不一样，它通过如 Gemini 这些多模态大模型来识别理解页面上的元素。

通俗的理解，就是它能 “看到” 整个页面的截图内容，自我判断哪里是按钮、哪里是输入框、该点哪里、该填什么。

这样就完美解决了，网站改版导致自动化脚本失效问题，不需要我们重新去找 XPath 或者 CSS 选择器。
